# Personal Information Manager

## 📌 Project Description
This is my first Python project. It is a simple program that stores and displays personal information entered by the user.

## 🚀 Features
- Stores user details like name, age, city, and hobby
- Takes user input for favorite food and color
- Displays information in a clean format
- Validates user input (no empty values)
- Calculates age in months
- Menu-driven program

## 🧠 What I Learned
- Variables and data types
- Input and output in Python
- Functions
- Loops (while loop)
- Conditional statements (if-else)
- Basic validation
- Dictionary usage
